# Prototype B: Micro-Scale Granular-Ferrofluid Cell

## Overview

The second prototype (paper Section 4.2) targets micro-scale cells (10-100 µm) compatible with fiber optic integration for scalable arrays.

## Status: FUTURE (requires Phase 2 completion)

## Specifications

| Parameter      | Value                                 | Notes                  |
| -------------- | ------------------------------------- | ---------------------- |
| Medium         | Granular composite + ferrofluid + ZIM |                        |
| Cell size      | 10-100 µm                             | Fiber-compatible       |
| Excitation     | RF or optical pulses (MHz)            | Through fiber          |
| Readout        | Laser diffraction + photodiodes       |                        |
| Quantum option | Optical coupling to Rb vapor cell     |                        |
| Target density | 1-10 Tb/cm³                           | With ZIM, 10 bits/mode |

## Prerequisites

Before attempting Prototype B:

- [ ] Prototype A validates Q > 100 in ferrofluid
- [ ] ZIM structures confirmed effective at macro scale
- [ ] Fabrication partner identified (cleanroom access)
- [ ] Fiber coupling simulated and validated

## Fabrication Approaches

1. **Focused Ion Beam (FIB)** — Gold nanoholes/pillars on fiber facets
2. **Electron-Beam Lithography (EBL)** — Silicon pillar arrays for ZIM
3. **Two-Photon Polymerization (TPP)** — 3D-printed honeycomb ZIM on fiber tips
4. **Nanoimprint / Transfer** — Metallic antennas or dielectric stacks

## Fiber Integration Options

- **Endpoint**: Cell terminates fiber as micro-resonator
- **In-line**: Cell embedded mid-fiber for through-point processing
- **Array**: Multiple fibers terminated in cell array with shared addressing
