Board D - AWG Drive / Ch B Reference, 1:1 Breadboard Footprint

Use this board only if you need a physical distribution point. The best Ch B reference is still direct BNC/coax from the AWG tee to Pico Ch B. Keep Board D far from Board A.

Route drive pair away from Board A.If wires cross RX, cross at 90 degrees.

Preferred AWG Tee Wiring

Pico AWG OUT Into BNC tee.
Tee branch 1 Direct BNC/coax to Pico Ch B. Use Ch B +/-500 mV.
Tee branch 2 Twisted pair or short coax to drive PZT.
Board D Only a distribution/dummy-load point if needed. Keep it left/back, far from Board A.

Dummy-Load Control Header

Normal AWG tee branch 2 to drive PZT.
Dummy load AWG tee branch 2 to a resistor/cap load instead of the PZT.
Disconnected drive AWG reference on Ch B only; drive PZT disconnected.
Success Ch A residual should drop when the physical drive path is removed.

Board M - Relay Mux / Arduino, 1:1 Breadboard Footprint

This board represents the digital/mux zone. Keep relay coil currents, USB ground, and Arduino wiring away from Board A. The only analog path leaving this zone should be the selected RX signal and its return, twisted together.

Selected RX signal leaves as a short twisted pair.Relay/Arduino power does not share Board A rails.
Mux Wire Destination
RX PZT signal leads Relay inputs for channels 5 and 6.
RX PZT return Route with signal as twisted pair. Tie to analog star only where required.
Mux common signal Board A RX input through 1k-10k series resistor.
Mux common return Board A analog star ground.
Keep Separate Reason
Arduino USB ground No relay/USB current through analog star path.
Relay coil supply Coil switching injects noise into long ground wires.
AWG drive pair Parallel routing capacitively injects carrier into RX.
Preamp input node Keep it only on Board A and as short as possible.

Bring-Up Checklist and Test Order

Power-Off Inspection

  • OPA pin 8 goes to +9 V.
  • OPA pin 4 goes to -9 V.
  • Battery midpoint is the Board A analog star ground.
  • 100 nF cap from +9 V to 0 V close to pin 8.
  • 100 nF cap from -9 V to 0 V close to pin 4.
  • 10 uF cap from +9 V to 0 V.
  • 10 uF cap from 0 V to -9 V, polarity correct.
  • Unused op amp B is wired as follower.
  • OPA input node has 1M to analog ground.
  • OPA output has 47R-100R before Pico Ch A.
  • AWG drive pair is far from Board A input.
  • Board A has no Arduino, relay, or AWG drive wires.

Multimeter Checks

+9 rail to analog ground +8 to +9.6 V
-9 rail to analog ground -8 to -9.6 V
+9 rail to -9 rail 16 to 19 V
OPA output, input grounded near 0 V, not near a rail
Pico Ch A at +/-200 mV no rail with input grounded
Pico Ch B range use +/-500 mV for AWG tee reference
Important: Ch B +/-50 mV railed in the dual-channel test. Ch B +/-500 mV was the usable range.

Test Sequence Before Real Data

Step Bench State Expected Result
1. Input short Board A input node tied to analog ground, AWG off. Ch A low RMS, no rail.
2. Layout feedthrough Input still grounded, AWG on at 10 mVpp. Ch A should not show large carrier/harmonics. If it does, layout/ground is coupling.
3. Mux termination Mux common terminated to analog ground; no RX PZT. Ch A remains quiet with AWG on.
4. Normal acoustic Mux common to Board A input; RX PZT selected. Ch A response appears; Ch B cancellation should reduce direct component.
5. Reference-cancel scan Normal acoustic, Ch B direct tee reference. Top cancelled bins should stop being mostly exact harmonics.
