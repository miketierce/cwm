# Community Data

Submit your completed worksheets and raw PicoScope data files here via pull request.

## How to contribute

1. Create a folder named `YYYY-MM-DD_your-name/` (e.g., `2026-04-15_smith-lab/`).
2. Include scanned/photographed Worksheets D.1–D.13 and the Experiment Log (D.17).
3. Include raw PicoScope waveform files (.psdata or .csv) for each experiment.
4. Submit via pull request. Use your Experiment Log as the PR description.

Every data point matters — see the Experiment Guide §D.19 for details.
