# Image Search Library

Place 16+ reference images (JPEG or PNG) in this directory for Experiment 13.

Any images will work — corrosion patterns, faces, product photos, symbols, etc.
For best results, choose visually distinct images. The tool computes a 64-bit
perceptual hash for each image and maps it to a rod/channel combination.

See the Experiment Guide (Experiment 13) for full instructions.
