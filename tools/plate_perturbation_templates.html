Pattern Design Rationale (from E38 3-condition results):
• Perturbation location matters more than mass — sites must sit near mode antinodes
• Each pattern targets different mode families of a free square plate (Chladni patterns)
• Pattern A: (1,1) + (2,2) antinodes — center + quarter-points
• Pattern B: (2,1) + (1,2) antinodes — edge midpoints
• Pattern C: (3,3) grid — higher-order mode antinodes at third-points
• Pattern D/E (duplicate): Diagonal — asymmetric modes, breaks (m,n)↔(n,m) degeneracy
• Pattern F: Anti-diagonal zigzag — complementary to D/E, maximizes coupling independence for 3-plate ESN. Apply to physical plate C.
• Pattern G (NEW): Asymmetric pentagon — all 5 sites in D₄ general position (no axis, no symmetry relation). Apply to physical plate E (replaces diagonal duplicate). Now 4 distinct plates for ESN.

Optimizer-Recommended Patterns (Patterns H–L):
These patterns were generated by the 3-PZT plate optimizer (plate_3pzt_pattern_optimizer.py) which maximizes resolved peak count — the number of individually resolvable spectral peaks after splitting degenerate (n,m)↔(m,n) mode pairs via perturbation symmetry breaking.
• Pattern H: Edge-parallel — 5 sites along y=40 mm line. Top scorer: 128 peaks, 56 splits (vs D: 111 peaks, 39 splits).
• Pattern I: Asymmetric diagonal — curved y(x) path breaking x↔y mirror. 126 peaks, 53 splits.
• Pattern J: R₂ quasi-random — Roberts (2018) low-discrepancy sequence. 125 peaks, 53 splits.
• Pattern K: Off-diagonal cluster — SE quadrant arc. 123 peaks, 50 splits.
• Pattern L: Scattered interior — Halton sequence, 7 sites. 122 peaks, 50 splits.
