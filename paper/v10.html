Wave-Coupled Ferromagnetic Oscillator Memory Architecture (WCFOMA): Spectral Eigenmode Memory and Computation in Acoustic Resonators

Michael Tierce Forward Arrow Solutions, Dallas, TX

Abstract

We introduce the Wave-Coupled Ferromagnetic Oscillator Memory Architecture (WCFOMA), a computing paradigm that encodes information in the acoustic eigenmode spectrum of physical resonators and performs associative recall through wave interference. Unlike conventional digital memory, which stores bits in spatial cells and shuttles them to a separate processor, WCFOMA stores data as frequency-domain patterns—each resonator’s mode spectrum constitutes both the memory and the computational substrate. We present validated simulation results for four candidate implementations spanning three physical substrates: ferrofluid acoustic cavities, ferroelectric photonic interferometers, and solid glass acoustic resonators. Our analysis reveals that the originally proposed ferrofluid approach is fundamentally limited by nanoparticle phase diffusion, which dominates 77.5% of the noise budget and yields negative SNR at room temperature. The glass acoustic resonator, by contrast, achieves 98.8 dB SNR, supports 9,380 thermally stable modes per rod at ±1 K stability, and delivers an estimated 154,000 bits of dynamic spectral capacity from a $63 prototype built entirely from commodity hardware. We derive information-theoretic capacity bounds using Shannon’s channel capacity theorem, validate perturbation encoding via Rayleigh’s spectral perturbation formula, and demonstrate Hopfield-compatible associative recall through spectral correlation. The glass resonator represents, to our knowledge, the first acoustic eigenmode memory architecture that can be prototyped without specialized equipment—a direct descendant of the mercury delay lines that powered UNIVAC I, reimagined for spectral rather than temporal encoding.

1. Introduction

1.1 The Memory Wall

Modern computing faces a fundamental architectural bottleneck. In 1978, John Backus identified the “von Neumann bottleneck”—the narrow channel between processor and memory that limits throughput regardless of processor speed [1]. Nearly two decades later, Wulf and McKee quantified the growing disparity between processor and memory performance, coining the term “memory wall” [2]. Today, data movement between DRAM and CPU accounts for the majority of both energy consumption and latency in large-scale computation, a problem that worsens with each process node shrink.

Emerging memory technologies—phase-change memory (PCM), resistive RAM (ReRAM), spin-transfer torque MRAM—have demonstrated in-memory computing capabilities by performing analog multiply-accumulate operations at the memory array itself [3]. However, these approaches remain fundamentally spatial: each bit occupies a physical cell, and computation requires explicit matrix-vector operations through crossbar arrays.

1.2 Wave-Based Memory: A Different Encoding

WCFOMA proposes an orthogonal approach: encode information in the frequency domain rather than in spatial cells. A physical resonator supporting $N$ acoustic eigenmodes can store information across all $N$ modes simultaneously. Each mode’s amplitude, phase, and frequency constitute independent information channels. Crucially, wave interference—a physical process that occurs at the speed of sound—performs the equivalent of a dot product across all modes in a single operation, enabling content-addressable recall without sequential memory access.

This is not a new idea in principle. Mercury delay line memory, used in UNIVAC I (1949) and EDSAC (1948), stored data as acoustic pulse trains circulating in tubes of mercury [4, 5]. Those systems used temporal encoding—bits were time-slotted pulses. WCFOMA uses spectral encoding—bits are eigenmode amplitudes and perturbation-induced frequency shifts. The distinction is profound: temporal encoding scales with tube length (one dimension), while spectral encoding scales with the number of resolvable modes (potentially thousands per resonator).

1.3 Scope and Contributions

This paper makes the following contributions:

  1. Architecture definition: We formalize WCFOMA as a memory and computation architecture based on eigenmode spectral encoding, perturbation-based data writing, and interference-based associative recall (Section 2).

  2. Rigorous noise analysis: We model six noise sources for the ferrofluid substrate and demonstrate that nanoparticle phase diffusion fundamentally limits the original proposal, contributing 77.5% of total noise power (Section 3).

  3. Glass resonator breakthrough: We identify solid glass rods as a superior substrate—eliminating phase diffusion entirely—and validate through simulation that a 6 mm × 150 mm borosilicate rod achieves 98.8 dB SNR and 9,380 thermally stable modes (Section 4).

  4. Technology comparison: We benchmark WCFOMA against seven conventional memory technologies (DRAM, SRAM, NAND, PCM, MRAM, ReRAM, magnonic logic) and three WCFOMA substrate variants on energy, density, latency, endurance, and compute locality (Section 5).

  5. Prototype specification: We provide a complete bill of materials ($63), five structured experiments, and quantitative predictions for a garage-buildable glass resonator prototype (Section 6).

2. Architecture

2.1 Eigenmode Memory

Consider a one-dimensional acoustic resonator of length $L$ with longitudinal wave speed $v$. The resonant frequencies are:

$$f_n = \frac{n \cdot v}{2L}, \quad n = 1, 2, 3, \ldots$$

Each mode $n$ is an independent harmonic oscillator with frequency $f_n$, quality factor $Q$, linewidth $\Delta f_n = f_n / Q$, and coherence time $\tau_n = Q / (\pi f_n)$. The mode amplitudes ${a_n}$ and phases ${\phi_n}$ constitute the memory state. For a resonator supporting $N$ resolvable modes, each with $b$ bits of amplitude resolution, the dynamic spectral capacity is:

$$C_{\text{dynamic}} = N \times b \quad \text{bits}$$

where $b$ is determined by the signal-to-noise ratio via Shannon’s channel capacity theorem [6]:

$$b = \frac{1}{2} \log_2(1 + \text{SNR})$$

2.2 Perturbation Encoding (Writing)

Data is written by modifying the resonator’s physical structure, which shifts mode frequencies according to the Rayleigh perturbation formula [7]. For a localized mass perturbation $\Delta m$ at position $x$ on a rod of total mass $M$:

$$\frac{\Delta f_n}{f_n} = -\frac{\Delta m}{2M} \sin^2!\left(\frac{n \pi x}{L}\right)$$

Each perturbation creates a unique pattern of frequency shifts across all modes—the shift depends on both the perturbation’s position (through the $\sin^2$ term) and its magnitude (through $\Delta m / 2M$). Multiple perturbations superpose linearly (in the small-perturbation limit), creating a high-dimensional spectral fingerprint that constitutes the stored data.

This encoding is physically analogous to the quartz crystal microbalance (QCM), where the Sauerbrey equation relates mass loading to frequency shift [8]. WCFOMA generalizes this to multi-mode operation, extracting information from the pattern of shifts across many modes rather than a single frequency shift.

Writing methods (in order of permanence):

  • Temporary: beeswax or clay blobs (removable, rewritable)
  • Semi-permanent: epoxy drops, tape segments
  • Permanent: notches, grinding, laser ablation

2.3 Interference Recall (Reading and Computing)

Readout measures the mode spectrum using a pair of piezoelectric transducers: one drives, one senses. A frequency sweep or impulse excitation followed by FFT reveals all mode frequencies simultaneously.

Associative recall exploits wave interference directly. Given an array of $R$ rods, each with a distinct perturbation pattern (stored data), driving all rods with a query spectrum causes each rod to resonate in proportion to the overlap between the query and its stored pattern. The rod whose pattern best matches the query produces the largest response—this is a physical implementation of a normalized correlation:

$$\text{correlation}_r = \frac{\vec{q} \cdot \vec{s}_r}{|\vec{q}| \cdot |\vec{s}_r|}$$

where $\vec{q}$ is the query spectral signature and $\vec{s}_r$ is rod $r$’s stored signature. This operation is mathematically equivalent to one step of Hopfield network dynamics [9], where the energy function:

$$E = -\frac{1}{2} \sum_{ij} w_{ij} s_i s_j$$

is minimized by the interference process. The network’s storage capacity follows the Amit-Gutfreund-Sompolinsky limit [10]:

$$P_{\max} = \alpha_c \cdot N, \quad \alpha_c \approx 0.138$$

For $N = 100$ modes, this permits $P_{\max} \approx 13$ stored patterns with reliable recall. Trinary encoding ($s_i \in {-1, 0, +1}$) increases capacity by approximately $2.1\times$.

2.4 Architecture Summary

Layer Function Physical Process
Storage Eigenmode amplitudes Driven acoustic modes
Write Perturbation encoding Mass attachment / removal
Read Spectral measurement Piezo drive + sense + FFT
Compute Associative recall Multi-rod wave interference

The key architectural distinction from conventional memory: there is no separation between storage and computation. The interference that reads the memory simultaneously computes the nearest-neighbor match. This is inherent to wave physics—superposition and interference are not engineered features but fundamental properties of the medium.

3. Noise Analysis and the Ferrofluid Limitation

3.1 The Original Proposal

The original WCFOMA concept proposed ferrofluid—a colloidal suspension of magnetic nanoparticles (~10 nm diameter) in a carrier fluid—as the acoustic medium. The appeal was clear: ferrofluids respond to external magnetic fields, enabling non-contact mode manipulation. The proposal assumed $Q \approx 500$, 10 modes per cavity, and 10 bits per mode, yielding 100 bits per 10 µm cell.

3.2 Six Noise Sources

Our simulation models six independent noise mechanisms for the ferrofluid substrate:

1. Thermal (Johnson-Nyquist) noise. Fundamental thermodynamic limit:

$$S_{\text{thermal}} = \frac{4 k_B T}{Z \cdot V}$$

where $Z$ is acoustic impedance and $V$ is cavity volume. At $T = 300$ K, $V = 10^{-15}$ m³ (a 10 µm cavity), this contributes modestly.

2. Shot noise. For optical readout with $N_\gamma$ photons:

$$S_{\text{shot}} = \frac{1}{N_\gamma}$$

At $N_\gamma = 10^6$ (the baseline assumption), shot noise is small. At $N_\gamma = 10^8$, it becomes negligible.

3. 1/f noise. Empirically observed in ferrofluids with corner frequency $f_c \approx 1$ kHz and amplitude $\alpha_{1/f} \approx 10^{-12}$.

4. Phase diffusion. This is the killer. Brownian rotational diffusion of magnetic nanoparticles randomizes the local magnetic susceptibility, causing stochastic phase drift in acoustic modes. The diffusion coefficient:

$$D_{\text{rot}} = \frac{k_B T}{\pi \eta d^3}$$

For 10 nm particles in carrier fluid ($\eta = 0.01$ Pa·s): $D_{\text{rot}} \approx 1.3 \times 10^7$ rad²/s. This is fast enough to destroy mode coherence on microsecond timescales.

5. Nonlinear mode mixing. Four-wave coupling between modes, proportional to $\chi_{\text{nl}} \cdot n^2$.

6. Quantization noise. ADC discretization: $\sigma_q^2 = \Delta^2 / 12$ for a 10-bit ADC.

3.3 Noise Budget

Our simulation computes the fractional contribution of each source to total noise power:

Source Fraction Notes
Phase diffusion 77.5% Dominant; inherent to liquid
1/f noise 11.8% Empirical; material-dependent
Nonlinear mixing 5.9% Grows with mode count
Thermal 3.4% Fundamental limit
Quantization 1.2% ADC-limited
Shot noise 0.2% At $10^6$ photons

Phase diffusion alone accounts for more than three-quarters of all noise. This is not an engineering problem to be optimized away—it is a fundamental property of storing coherent acoustic modes in a medium containing freely rotating nanoparticles.

3.4 Ferrofluid SNR Results

Baseline ($Q = 500$, $V = 10^{-15}$ m³, $10^6$ photons, $\eta = 0.01$ Pa·s):

  • SNR = −6.5 dB (below 0 dB—noise exceeds signal)
  • Usable modes: 0
  • Bits per measurement: 0
  • The original claim of 10 bits/mode overestimates by $\sim 4.4\times$

Mitigated (gel immobilization $\eta = 1.0$ Pa·s, $10^8$ photon readout):

  • SNR = +13.5 dB
  • Usable modes: 10
  • Total bits: 22.8 (2.28 bits/mode)
  • Still requires expensive laser and sacrifices ferrofluid’s key advantage (field responsiveness)

The ferrofluid approach is not viable for practical memory. The noise physics are unfavorable at room temperature in micron-scale cavities.

4. Glass Acoustic Resonator

4.1 Motivation: Eliminating Phase Diffusion

The noise analysis of Section 3 points to a clear solution: use a solid medium. A solid has no Brownian motion, no nanoparticle rotation, and no phase diffusion. Glass—one of the oldest engineered materials—is an acoustic resonator with well-characterized properties, quality factors orders of magnitude higher than ferrofluid, and a procurement cost measured in dollars rather than hundreds of dollars.

The historical precedent is exact. Mercury delay line memory [4, 5] stored acoustic pulses in a liquid medium. The transition from liquid mercury to solid glass mirrors the transition from ferrofluid to glass in WCFOMA—eliminating the liquid-phase noise mechanisms while preserving the acoustic physics.

4.3 Mode Spectrum

For the reference design—a 6 mm diameter × 150 mm borosilicate rod:

$$f_1 = \frac{5{,}500}{2 \times 0.15} = 18{,}333 \text{ Hz}$$

Mode spacing equals $f_1 = 18{,}333$ Hz. The coherence time of the fundamental mode:

$$\tau_1 = \frac{Q}{\pi f_1} = \frac{10{,}000}{\pi \times 18{,}333} = 0.174 \text{ s}$$

Higher modes have proportionally shorter coherence times ($\tau_n = \tau_1 / n$), but even mode $n = 100$ retains $\tau_{100} = 1.74$ ms—ample for measurement.

4.4 SNR Analysis

The only noise source for a solid glass rod at room temperature is thermal phonon noise. The effective spring constant for mode $n = 1$:

$$k_{\text{eff}} = m_{\text{eff}} \cdot \omega_1^2$$

where $m_{\text{eff}} = M/2$ (half the rod mass for the fundamental mode). For our reference rod ($M = \rho \cdot \pi r^2 L = 2{,}230 \times \pi \times (0.003)^2 \times 0.15 = 9.46 \times 10^{-3}$ kg):

$$m_{\text{eff}} = 4.73 \times 10^{-3} \text{ kg}$$

$$k_{\text{eff}} = m_{\text{eff}} (2 \pi f_1)^2 = 4.73 \times 10^{-3} \times (2\pi \times 18{,}333)^2 = 6.27 \times 10^7 \text{ N/m}$$

Signal energy at 1 nm drive amplitude:

$$E_{\text{signal}} = \frac{1}{2} k_{\text{eff}} A_{\text{drive}}^2 = \frac{1}{2} \times 6.27 \times 10^7 \times (10^{-9})^2 = 3.14 \times 10^{-11} \text{ J}$$

Thermal noise energy:

$$E_{\text{thermal}} = k_B T = 1.38 \times 10^{-23} \times 300 = 4.14 \times 10^{-21} \text{ J}$$

$$\text{SNR} = \frac{E_{\text{signal}}}{E_{\text{thermal}}} = \frac{3.14 \times 10^{-11}}{4.14 \times 10^{-21}} = 7.58 \times 10^{9}$$

$$\text{SNR}{\text{dB}} = 10 \log{10}(7.58 \times 10^9) = 98.8 \text{ dB}$$

$$b = \frac{1}{2} \log_2(1 + 7.58 \times 10^9) = 16.4 \text{ bits/mode}$$

Phase diffusion: 0%. The glass is a solid. There are no nanoparticles. There is no Brownian motion. The noise floor is set by the thermodynamic limit—and the signal is ten billion times above it.

4.5 Thermal Mode Stability

Not all modes are usable. Thermal drift shifts mode frequencies, and if adjacent modes drift into each other, they become unresolvable. The stability criterion requires that thermal drift plus intrinsic linewidth remain smaller than mode spacing:

$$n_{\max} = \left\lfloor \frac{1}{2 \alpha \Delta T + 1/Q} \right\rfloor$$

For borosilicate at $\pm 1$ K:

$$n_{\max} = \left\lfloor \frac{1}{2 \times 3.3 \times 10^{-6} \times 1 + 1/10{,}000} \right\rfloor = \left\lfloor \frac{1}{6.6 \times 10^{-6} + 10^{-4}} \right\rfloor = \left\lfloor \frac{1}{1.066 \times 10^{-4}} \right\rfloor = 9{,}380$$

Total dynamic capacity:

$$C = 9{,}380 \times 16.4 \approx 154{,}000 \text{ bits} \approx 19 \text{ kB}$$

This is the capacity of a single 150 mm glass rod, maintained at $\pm 1$ K stability, with 1 nm acoustic drive. Better temperature control or fused silica ($Q = 100{,}000$, $\alpha = 0.55 \times 10^{-6}$/K) would dramatically increase $n_{\max}$.

4.6 Perturbation Encoding Capacity

Beyond dynamic spectral capacity (mode amplitudes), perturbation encoding stores data in the structure of the rod itself. Each mass perturbation encodes:

  • Position: $\log_2(R_{\text{pos}})$ bits, where $R_{\text{pos}}$ is the number of distinguishable positions
  • Mass: $\log_2(M_{\text{levels}})$ bits

For $K$ perturbation blobs on a rod with $R_{\text{pos}} = 100$ distinguishable positions and $M_{\text{levels}} = 8$ mass levels:

$$C_{\text{pert}} = K \times (\log_2 100 + \log_2 8) = K \times (6.64 + 3) = K \times 9.64 \text{ bits}$$

With 10 blobs: 96.4 bits. With 50 blobs: 482 bits. This data is non-volatile—it persists as long as the physical perturbations remain attached.

5.3 Where WCFOMA Fits

WCFOMA is not competitive with DRAM or NAND on raw density or speed. A 150 mm glass rod storing 19.2 kB occupies far more volume than a DRAM chip storing gigabytes. This is expected—WCFOMA is not a replacement for conventional memory.

WCFOMA’s value proposition is compute locality. In conventional architectures, associative recall (nearest-neighbor search, pattern completion) requires:

  1. Reading all stored patterns from memory
  2. Computing similarity metrics in the processor
  3. Selecting the best match

This is O($P \times N$) in time and energy, where $P$ is the number of stored patterns and $N$ is the pattern dimension. In WCFOMA, wave interference performs step 2 in a single operation at the speed of sound—the entire rod array resonates simultaneously, and the best match emerges as the rod with the largest response amplitude. The computation time is independent of $P$ (it depends only on the acoustic propagation time, ~microseconds).

This makes WCFOMA potentially relevant for:

  • Edge AI inference: Small associative memory for pattern recognition without a GPU
  • Sensor preprocessing: Acoustic signature matching (vibration fingerprinting, defect detection)
  • Analog computing research: A tangible, buildable platform for studying wave-based computation
  • Education: A $63 demonstration of eigenmode physics, information theory, and associative memory

5.4 The Ferroelectric Photonic Path

For applications requiring higher density and integration, the ferroelectric photonic variant offers a path to chip-scale WCFOMA. Using HZO (Hf₀.₅Zr₀.₅O₂) ferroelectric films on SiN waveguides in a Mach-Zehnder interferometer (MZI) configuration [13]:

  • $\Delta n \approx 10^{-3}$ (HZO refractive index change per polarization state)
  • Trinary encoding ($P_{\text{up}}$, $P_{\text{down}}$, $P_{\text{zero}}$): up to $\log_2(3) = 1.58$ bits/cell (requires interaction length $\gtrsim 775$ µm for full $\pi$ phase shift; shorter devices achieve proportionally fewer bits)
  • Write energy: 0.1–1 nJ per cell depending on device area ($V_c = 5$ V, 100 ns switching)
  • Read energy: ~0.13 fJ (single photon-level detection)
  • Endurance: $10^5$ cycles (HZO fatigue limit)
  • Retention: ~12 hours (HZO depolarization)

A crossbar array of MZI cells could implement a Hopfield network in photonic hardware, with each cell’s ferroelectric state constituting a synaptic weight. This requires cleanroom fabrication and is currently at simulation stage, but the materials (HZO, SiN) are CMOS-compatible—no exotic substrates required.

Integrated photonic memory using phase-change materials (Ge₂Sb₂Te₅) on waveguides has been demonstrated experimentally [14], establishing the viability of non-volatile photonic memory cells. The ferroelectric approach trades PCM’s higher endurance concerns for CMOS compatibility and potentially faster switching.

6.2 Experimental Plan

We propose five experiments of increasing complexity, each with explicit success and kill criteria:

Experiment 1: Mode spectrum measurement. Epoxy one PZT disc (drive) to one rod end, another (sense) to the opposite end. Rod rests on foam isolation mat. Sweep drive frequency from 1 kHz to 500 kHz (or tap rod and compute FFT of ring-down signal). Prediction: At least 5 resonant peaks matching $f_n = n \times 18{,}333$ Hz within 5% accuracy. Kill criterion: No peaks above noise floor → check piezo coupling, try direct contact.

Experiment 2: Q factor measurement. Impulse-excite rod, record decay envelope. $Q = \pi f \tau$, where $\tau$ is the 1/e amplitude decay time [12]. Prediction: $Q = 5{,}000$–$10{,}000$ for borosilicate. Kill criterion: $Q < 100$ → severe coupling losses; reduce epoxy thickness or clamp at a node.

Experiment 3: Perturbation detection. Measure bare rod spectrum. Attach 0.1 g beeswax blob at $L/4$. Remeasure. Compare. Prediction: ~48 Hz shift in mode 1. Different positions ($L/4$ vs. $L/3$) yield different shift patterns per the Rayleigh formula. Kill criterion: Shifts smaller than linewidth → use larger mass (0.5 g) or more sensitive readout.

Experiment 4: Pattern discrimination. Two rods with different perturbation patterns (A: wax at $L/4 + L/2$; B: wax at $L/3 + 2L/3$). Measure both spectra and compute spectral correlation. Prediction: Same-pattern correlation $> 0.95$; cross-pattern correlation $< 0.5$. Kill criterion: Indistinguishable patterns → increase mode count or mass variation.

Experiment 5: Associative recall. Three or more rods with distinct patterns. Drive all with a query spectrum. Measure which rod responds most strongly. Prediction: Correct rod identified with $> 90\%$ accuracy over 10 trials. Kill criterion: Cross-talk prevents discrimination → add acoustic isolation between rods.

6.3 What Success Looks Like

If Experiments 1–3 succeed, we will have demonstrated:

  • Multi-mode acoustic resonance in a glass rod (eigenmode memory substrate)
  • Measurable Q factors validating simulation predictions
  • Perturbation-induced frequency shifts matching the Rayleigh formula (data writing)

If Experiments 4–5 succeed, we will have demonstrated:

  • Spectral fingerprint discrimination (content-addressable storage)
  • Interference-based associative recall (computation via wave physics)

This would constitute, to our knowledge, the first experimental demonstration of a spectral eigenmode memory architecture in a garage-buildable device.

7. Discussion

7.1 What We Know

Our simulations are grounded in well-established physics:

  • Acoustic mode frequencies follow $f_n = nv/2L$ (textbook wave mechanics)
  • Glass Q factors are extensively characterized in the ultrasonic testing literature [11, 12]
  • Rayleigh perturbation theory has been validated in millions of quartz crystal microbalances [8]
  • Hopfield associative memory capacity is a solved problem in statistical mechanics [9, 10]
  • Shannon capacity provides a rigorous upper bound on information per mode [6]

The glass resonator simulation uses no free parameters—every input (wave speed, density, Q factor, thermal expansion) comes from published material databases or manufacturer datasheets.

7.2 What We Don’t Know

Several critical questions remain unresolved and require experimental validation:

  1. Piezo-to-glass coupling efficiency. Our simulation assumes ideal transduction. Real epoxy bonds will introduce losses that reduce effective Q and potentially create spurious modes. The magnitude of this loss is not known a priori—it must be measured (Experiment 2).

  2. Multi-mode readout fidelity. We model modes as perfectly independent oscillators. In reality, nonlinear coupling between modes, boundary reflections, and transducer bandwidth limitations may degrade the ability to resolve high-order modes. The number of practically usable modes may be substantially less than the theoretical 9,380.

  3. Perturbation reproducibility. The Rayleigh formula assumes point masses. Real beeswax blobs have finite extent, and the exact coupling depends on contact area, stiffness, and blob geometry. The resolution of perturbation encoding may be lower than calculated.

  4. Environmental noise. Acoustic isolation from ambient vibration (footsteps, HVAC, traffic) is not modeled. The foam mat provides some decoupling, but environmental noise may limit practical SNR below the thermal limit.

  5. Scaling. A single rod stores ~19 kB. Scaling to meaningful capacities requires rod arrays, which introduces cross-talk, impedance matching, and multiplexing challenges that are not addressed in this work.

7.3 Honest Assessment

WCFOMA in its glass resonator form is not a competitor to silicon memory technology. It is slower, larger, and stores less data. Its value is conceptual and educational: it demonstrates that eigenmode spectral encoding and interference computation are physically realizable with commodity materials, and it provides a concrete platform for exploring wave-based computing ideas that may eventually find application in specialized domains (acoustic sensing, analog AI, neuromorphic computing) or in more advanced substrates (photonic, magnonic).

The ferrofluid approach, as originally conceived, does not work. Phase diffusion is a fundamental obstacle at room temperature. We report this negative result because it is important: the physics is beautiful but the engineering is intractable without cryogenic temperatures or gel immobilization, both of which negate the medium’s advantages.

The ferroelectric photonic approach remains promising but requires fabrication capabilities beyond a garage lab. It is the most likely path to a practical WCFOMA device, leveraging the established silicon photonics ecosystem.

7.4 Relationship to Existing Work

WCFOMA’s closest relatives in the literature are:

  • Acoustic delay line memory [4, 5]: Same physical medium (acoustic waves), but temporal rather than spectral encoding. WCFOMA’s multi-mode approach is the spectral generalization.
  • Quartz crystal microbalance (QCM) [8]: Uses the same Rayleigh perturbation physics for mass sensing. WCFOMA extends QCM from single-mode measurement to multi-mode information encoding.
  • Photonic neural networks (Shen et al. 2017): Use optical interference for matrix-vector multiplication. WCFOMA applies the same principle in acoustic rather than optical domains.
  • In-memory computing with resistive devices [3]: Shares the goal of co-locating storage and computation, but uses resistive switching rather than wave interference.
  • Geometry-invariant resonant cavities [15]: Zero-index metamaterial cavities with shape-independent eigenfrequencies provide theoretical support for robust mode structures in arbitrarily shaped resonators—relevant to future WCFOMA miniaturization.

8. Conclusion

We have introduced WCFOMA, a memory and computation architecture that encodes data in the eigenmode spectrum of acoustic resonators and computes via wave interference. Through systematic simulation of four device variants across three substrates, we have identified the glass acoustic resonator as the most promising near-term implementation:

  • 98.8 dB SNR from thermal-noise-limited operation (no phase diffusion)
  • 9,380 thermally stable modes per 150 mm borosilicate rod at ±1 K
  • ~154,000 bits dynamic spectral capacity (~19 kB per rod)
  • $63 bill of materials using exclusively commodity components
  • 0% phase diffusion (vs. 77.5% for ferrofluid)
  • Five structured experiments with quantitative predictions

We have also provided an honest negative result: the originally proposed ferrofluid substrate is fundamentally limited by nanoparticle phase diffusion and cannot achieve positive SNR at room temperature without mitigation measures that eliminate its key advantages.

The glass resonator prototype, if validated experimentally, would demonstrate that spectral eigenmode memory is physically realizable—seventy-five years after mercury delay lines first stored data as acoustic waves in UNIVAC I, the same physics may find new expression as a platform for wave-based associative computing.

Appendix B: Derivation of Key Equations

B.1 Mode Frequency

For a free-free rod of length $L$ with longitudinal wave speed $v$, standing wave boundary conditions require:

$$L = n \frac{\lambda}{2} \implies \lambda_n = \frac{2L}{n} \implies f_n = \frac{v}{\lambda_n} = \frac{nv}{2L}$$

B.2 Rayleigh Perturbation

For a small mass $\Delta m$ attached at position $x$ on a rod vibrating in mode $n$, the kinetic energy perturbation is proportional to the square of the mode shape at the attachment point. The mode shape for longitudinal vibrations is $\psi_n(x) = \sin(n\pi x / L)$, giving:

$$\frac{\Delta f_n}{f_n} = -\frac{\Delta m}{2M} |\psi_n(x)|^2 = -\frac{\Delta m}{2M} \sin^2!\left(\frac{n\pi x}{L}\right)$$

The negative sign indicates that added mass lowers the frequency (increased inertia).

B.3 Shannon Capacity per Mode

Each acoustic mode is an additive white Gaussian noise channel with signal power $P_s = \frac{1}{2} k_{\text{eff}} A^2$ and noise power $P_n = k_B T$. Shannon’s theorem gives the capacity:

$$C = \frac{1}{2} \log_2(1 + \text{SNR}) \quad \text{bits per measurement}$$

The factor of $\frac{1}{2}$ accounts for real-valued (amplitude-only) measurement rather than complex (amplitude + phase).

B.4 Thermal Mode Limit

Two adjacent modes $n$ and $n+1$ have frequencies $f_n$ and $f_{n+1} = f_n + f_1$. Thermal drift shifts both by $\alpha \Delta T \cdot f_n$. The modes remain resolvable if the shift is less than half the spacing minus the linewidth:

$$2 f_n \alpha \Delta T < f_1 - \frac{f_n}{Q}$$

At the maximum mode number: $2 n_{\max} f_1 \alpha \Delta T = f_1 - n_{\max} f_1 / Q$, giving:

$$n_{\max} = \frac{1}{2\alpha\Delta T + 1/Q}$$
